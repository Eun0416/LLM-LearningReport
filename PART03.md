# 용어 정리

### 맞춤형 GPT (Custom GPT)
- 특정 목적이나 사용자의 요구에 맞춰 기능과 지식을 특화시킨 ChatGPT의 맞춤형 버전.
- 유료 옵션으로 사용자 스스로 생성하거나 기존의 것을 탐색하여 활용할 수 있다.

### GPT 탐색 (GPT Explore)
- ChatGPT 플랫폼 내에서 다양한 주제에 특화된 맞춤형 GPT를 검색하고 찾아볼 수 있는 기능.

### 확장 프로그램 (Brower Extension)
- 웹 브라우저의 기능을 확장하거나 사용자 인터페이스를 변경하기 위해 개발된 소프트웨어.
- 'Superpower ChatGPT'와 같이 ChatGPT 사용 환경을 개선하는 데 활용한다.

### API (Application Programming Interface)
- 소프트웨어 애플리케이션들이 서로 통신하고 데이터를 교환할 수 있도록 정의된 규칙과 인터페이스.
- 맞춤형 GPT가 외부 서비스나 도구와 연동하기 위해 사용될 수 있다.

### IMRaD 구조
- 과학 기술 논문에서 널리 사용되는 표준화된 형식으로, 서론(Introduction), 방법(Methods), 결과(Results), 논의(Discussion)의 네 부분으로 구성된다.

### 프롬프트 (Prompt)
- ChatGPT와 같은 AI 모델에게 특정 응답이나 작업을 요청하기 위해 사용자가 입력하는 텍스트.

### 프롬프트 관리자 (Prompts Manager)
- 'Superpower ChatGPT' 확장 프로그램의 기능으로, 자주 사용하는 프롬프트를 저장, 분류 및 재사용할 수 있도록 관리하는 도구.

### 채팅 아카이브 (Chat Archive)
- ChatGPT에서 사용하지 않지만 삭제하고 싶지 않은 대화를 보관하는 기능.
- 보관된 대화는 목록에서 숨겨지지만 필요할 때 다시 확인할 수 있다.

### 폴더 (Folder- Superpower ChatGPT)
- 'Superpower ChatGPT' 확장 프로그램에서 제공하는 기능으로, ChatGPT 대화 내용을 주제별 또는 프로젝트별로 분류하여 정리하고 관리할 수 있도록 한다.
