# 용어 정리

### ChatGPT (ChatGPT App)
- OpenAI에서 제공하는 독립 실행형 데스크탑 애플리케이션으로, 웹 브라우저 없이 ChatGPT를 사용할 수 있도록 합니다.

### GPT 탐색 (GPT탐색)
- ChatGPT 앱 내에서 사용자들이 다른 사용자들이 만들고 공유한 다양한 맞춤형 GPT를 찾아 활용할 수 있는 기능이다.

### GPT 만들기
- '내 GPT' 메뉴에서 새로운 맞춤형 GPT를 생성하는 과정을 시작하는 옵션입니다.

| 옵션 | 설명 |
|-|-|
| 만들기 (Create) | '새 GPT' 설정 화면의 첫번재 탭<br>GPT의 기본적인 역할과 목적을 설명하는 프롬프트를 입력하는 곳 |
| 구성 (Configure) | '새 GPT' 설정 화면의 두 번째 탭<br>GPT의 이름, 설명, 지침, 대화 스타터, 지식, 기능 등을 세부적으로 설정하는 곳 |
| 지침 (Instructions) | GPT가 따라야 할 규칙, 응답 스타일, 대화 방식 등을 명시하는 텍스트<br>GPT의 행동을 제어하는 핵심 요소|
| 대화 스타터 (Conversation Starters) | 사용자가 GPT와 대화를 시작할 때 참고할 수 있도록 미리 제시되는 예시 질문 또는 프롬프트 |
| 지식 (Knowledge) | GPT가 답변을 생성할 때 참고할 수 있도록 사용자가 업로드하는 문서, 데이터 파일 등의 외부 정보 |
| 기능 (Capabilities) | GPT가 활용할 수 있는 웹 검색, 이미지 생성 (DALE-E), 코드 인터프리터 등의 도구 활성화 옵션 |
| 미리보기 (Preview) | '새 GPT' 설정 과정에서 사용자가 만든 GPT가 실제로 어떻게 작동하는지 테스트해 볼 수 있는 화면 |

### GPT 스토어 (GPT Store)
- 사용자들이 자신이 만든 맞춤형 GPT를 공유하고 다른 사람들이 만든 GPT를 공유하고 다른 사람들이 만든 GPT를 찾아 사용할 수 있는 플랫폼입니다.
