# 용어 정리

### 이미지 그리기
- ChatGPT 기능을 사용하여 사용자의 프롬프트를 바탕으로 이미지 생성하는 기능.

### 캔버스(Canvas)
- ChatGPT 환경 내에서 그림 그리기, 글쓰기 코딩 등 다양한 창작 활동을 통합적으로 지원하는 기능.

### 이미지 이해 능력
- 이미지 안에 있는 글자, 차트, 표, 기호 등을 정확하게 인식하고 해석하는 인공지능의 능력.

### 프롬프트 (Prompt)
- 인공지능 모델에게 특정 작업(이미지 생성, 글쓰기 등)을 수행하도록 지시하는 입력 문구 또는 설명.

### 스타일 키워드
- 이미지 생성 프롬프트에서 특정 예술적 화풍, 분위기, 색감 등을 지정하기 위해 사용하는 단어 또는 구문.
- ex: 극사실주의, 사이버펑크, 지브리스타일

### 인페이팅 (Inpainting)
- 이미지의 특정 영역을 선택하여 해당 부분을 수정하거나 새로운 요소를 추가하는 이미지 편집 기술.

### 미드저니 스타일 (MidJourney Style)
- 디지털 아트 생성 AI 플랫폼인 미드저니 특유의 시각적 미학을 지칭하는 용어
- 현실과 환상이 섞인 고해상도 질감과 회화적인 표현이 특징이다.

### 다양한 글 편집 도구
| 기능 이름 | 설명 |
|-|-|
| 이모지 추가 | 문단 앞에 이모지를 삽입하여 시각적으로 강조하거나 감정적 뉘앙스를 더함.<br>개인화에 유용함. |
| 마지막으로 다듬기 | 현재 문단을 검토하거나 마무리 다듬기 대상임을 표시함.<br>협업 문서에서 유용함.|
| 독해 수준 | 문장의 난이도를 분석하여 독해가 쉬운지 어려운지 판단하고 표시함.<br>교육/에디터 환경에 적합함.|
| 길이 조절 | 문단의 길이를 자동으로 줄이거나 늘려줌.<br>간결하게 만들거나 자세히 풀어 쓰는 데 도움됨.|
| 편집 제안 | 문법이나 스타일에 대한 수정 제안을 ChatGPT가 자동으로 제공해 수정이 가능하며 실시간 문서 교정에 적합함.|

### 캔버스의 코드 편집 기능
| 기능 이름 | 주요 목적 | 작동 방식 및 예시 | 활용 상황 예시 |
|-|-|-|-|
| 댓글 추가<br>(Comment) | 코드나 문서에 주석(comments)과 피드백 제공| if (x==null) 구문에 대해 "널 체크 더 안전하게 할 수 있음" 같은 댓글 추가 | 협업 중 코드 설명 필요할 때|
| 로그 추가<br>(Add logs) | 디버깅<br>실행 흐름 추가| 함수 시작 시 console.log("Start Func") 자동 삽입 등 | 비동기 코드 흐름 파악<br>예외 추적|
| 버그 수정<br>(Fix bugs) | 코드 오류 자동 진단 및 수정 | for i in range(len(arr) →<br>for i, val in enumerate(arr)로 변경| 오타, 논리 오류 빠르게 고치고 싶을 때|
| 특정 언어로 포트<br>(Port to specific language) | 코드 이식성 향상 | Python → JavaScript 또는<br>Java → Kotlin 등으로 자동 변환 | 다국적 플랫폼 개발, 타 언어로 기능 옮겨야 할 때 |
| 코드 검토<br>(Code review) | 코드 품질 평가 및 개선 제안| "변수명이 모호함", "중복 있음", "복잡도 높음" 등 자동 피드백 제공 | 코드 제출 전 품질 점검, 리뷰 시간 절약하고 싶을 때|

### 캔버스 코드 뷰의 여러 메뉴
| 메뉴 이름 | 기능 설명 |
|-|-|
| 변경 사항 표시<br>(Show changes) | 현재 코드와 이전 버전 간의 변경 내용을 색상(diff)으로 비교해서 보여줌|
| 이전 버전<br>(Previous version) | 이전에 저장된 코드 상태로 돌아갈 수 있음 |
| 이후 버전<br>(Next version) | 되돌아간 상태에서 다시 최신 코드 상태로 이동 가능|
| 복사<br>(Copy) | 현재 코드 전체를 한 번에 클립보드에 복사|
| 공유하기<br>(Share) | 읽기 전용 링크를 생성해 다른 사람과 코드 내용을 공유할 수 있음 |
| 결과물 미리 보기<br>(Preview ouput) | 코드 실행 또는 문서의 출력 결과를 시각적으로 확인 |
