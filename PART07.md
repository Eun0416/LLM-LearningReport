# 용어 정리

### ChatGPT '작업' (tasks)
- ChatGPT의 기능 중 하나
- 사용자가 지정한 스케줄에 따라 주기적으로 특정 작업을 자동으로 수행하도록 예약할 수 있는 기능.
- 현재는 기능을 제공하지 않음.

### GPT-4o 일정 예약 베타
- ChatGPT의 특정 모델
- '작업' 기능과 같은 일정 예약 및 자동 반복 작업을 지원하는 베타 서비스 모델.

### 프롬프트 (Prompt)
- LLM(Large Language Model)에 사용자의 의도를 전달하기 위해 입력하는 명령, 질문 또는 지시문.
- AI의 응답 품질을 결정하는 중요한 요소

### 생성형 AI (Generative AI)
- 텍스트, 이미지, 코드 등 새로운 콘텐츠를 생성할 수 있는 인공지능 기술.

### LLM (Large Language Model)
- 대규모 텍스트 데이터로 학습되어 인간의 언어를 이해하고 생성하는 능력을 가진 인공지능 모델.

### Prompt Engineering
- LLM으로부터 원하는 결과를 얻기 위해 프롬프트를 효과적으로 설계하고 개선하는 기술.

### 지시형 프롬프트 (Instruction Prompt)
- AI에게 특정 행동이나 작업을 직접 명령하는 형태의 프롬프트.

### 질문형 프롬프트 (Question Prompt)
- AI에게 정보나 설명을 요청하는 형태의 프롬프트.

### 역힐 기반 프롬프트 (Role-based Prompt)
- AI에게 특정 역할이나 페르소나를 부여하여 해당 역할에 맞는 톤과 스타일로 응답하도록 유도하는 프롬프트.

### 샘플 포함형 프롬프트 (Few-shot Prompt)
- 프롬프트에 몇 가지 예시 질문과 답변을 함께 제공하여 AI가 특정 패턴이나 스타일을 따르도록 유도하는 방식.

### Zero-shot 프롬프팅
- 어떠한 예시도 제공하지 않고 프롬프트만으로 AI에게 작업을 요청하는 방식.

### One-shot 프롬프팅
- 하나의 예시 질문과 답변을 제공하고 작업을 요청하는 방식.

### Chain of Thought 프롬프트
- AI에게 단계별 사고 과정을 거쳐 결론에 도달하도록 유도하는 프롬프트 방식
- ex: "단계별로 생각해 봐" 또는 "step by step"

### GPT 탐색 (GPT Explorer)
- ChatGPT 플랫폼 내에서 다양한 목적과 기능에 특화된 사용자 맞춤형 GPT를 찾아보고 상용할 수 있는 기능.

### SlidesGPT
- AI 기반으로 프레젠테이션 슬라이드를 자동으로 생성해주는 외부 유료 서비스.
- ChatGPT와 연동하여 사용할 수 있는 기능.

### Felo (felo.ai)
- ChatGPT 기술 기반으로 프레젠테이션 등 다양한 콘텐츠 생성을 지원하는 AI 플랫폼.

### Gamma (Gamma.app)
- 아이디어 또는 텍스트를 기반으로 멋진 프레젠테이션, 문서, 웹 사이트 등을 자동으로 생성해주는 생성형 AI 도구 플랫폼.
- '텍스트로 붙여넣기' 또는 '생성' 기능을 통해 슬라이드를 만들 수 있다.
