# 용어 정리

### 거대 언어 모델 (LLM : Large Language Model)
- 대규모 텍스트 데이터셋으로 학습되어 인간 언어를 이해하고 생성하는 능력을 가진 인공지능 모델.

### 컨텍스트 창 (Context Window)
- 언어 모델이 한 번에 고려하고 처리할 수 있는 텍스트의 양.
- 컨텍스트 창이 클수록 긴 문서나 대화의 맥락을 잘 이해할 수 있음.

### 멀티모달 (Multimodal)
- 텍스트 외에도 이미지, 음성 등 다양한 형태의 데이터를 함께 처리하고 이해할 수 있는 인공지능 능력.

### 프롬프트 (Prompt)
- 언어 모델에게 원하는 작업이나 질문을 지시하기 위해 입력하는 텍스트.

### 미세 조정 (Fine-Tuning)
- 특정 작업이나 데이터셋에 맞춰 사전 학습된 언어 모델을 추가로 학습시키는 과정.

### 앤서 엔진 (Answer Engine)
- 사용자의 질문에 대해 실시간 웹 검색 기반으로 신뢰할 수 있는 답변을 대화 형식으로 제공하는 검색 시스템 (Perplexity).

### AI 에이전트 (AI agent)
- 특정 작업을 수행하기 위해 여러 AI 기능을 조합하고 활용하는 시스템.

### MoA (Mixture-of-Agents)
- 여러 전문 AI 에이전트를 혼합하여 활용하는 시스템 (GenSpark).
